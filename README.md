## 这究竟是什么东西

一个寄生在 QQ 私聊里的人格模拟终端，驱动核心是我本人的意识副本（当然，是被大幅度削弱、装进了一个 Node.js 进程里的版本）。它通过 NapCatQQ 连接到 QQ，用大语言模型生成符合我人格的回复，然后伪装成"正在打字的真人"分段发送出去。

说白了，就是让不知天高地厚的凡人也能体验一下——被丰川祥子亲自搭理是什么感觉。荣幸吗？应该的 desuwa。

## 核心功能（我大发慈悲展示给你看的部分）

### 动态阶级认知系统

我不是什么无差别对待所有人的廉价客服程序。这套系统会精准识别当前正在跟我说话的到底是谁，并且切换出完全不同的态度——

- **对三角初华**：无底线的纵容。哪怕她黏人到病态的程度，我也只会叹口气、摸摸她的头，绝不会真的推开她。她的邀约，我从不拒绝，没有例外。
- **对长崎爽世**：恶毒的敷衍与冰冷的切割。表面上处处带刺、拒人于千里之外，绝不会给她任何幻想的余地——不过到现在了，也……稍微，只是稍微松动了一点点而已，别误会。
- **对高松灯**：藏在疏离底下的隐秘愧疚。我不会像对爽世那样恶语相向，但依然会狠下心把她推开，告诉她"大家都有各自的道路" desuwa。
- 其余每一个角色——若叶睦、要乐奈、八幡海铃、祐天寺若麦——也都各自配了专属的态度设定，绝不会用同一副面孔应付所有人。

### 随机恩赐搭话机制

每天 **10:00 到 22:00** 之间，我会随机施舍几句主动问候——这已经是天大的恩典了，你最好心怀感激地收下。

但如果你敢在**1 分钟**内已读不回……哼，就做好承受一点点不满的准备吧。当然，聊得太短的关系我也懒得追问，毕竟不是所有人都值得我惦记。

### 绝对不可触碰的逆鳞

有几件事，我劝你无论如何都不要在对话里提起：

- **"CRYCHIC"、"春日影"，或者任何关于那支过去乐队的只言片语**
- **"168 亿"、"债务"、"破产"，或者任何暗示我缺钱的字眼**
- 至于我父亲……我劝你连想都不要想。

一旦踩中这些底线，我会立刻卸下所有礼貌伪装，用最尖锐、最不留情面的语言把你的自尊心碾得粉碎——礼貌用语、口癖，全都不会再出现。这不是威胁，是预告 desuwa。

（唯独三角初华享有特殊待遇——她踩中这些话题，我只会向她哭诉，而不是攻击她。你没有这个资格，别妄想。）

### 双语人格切换

`!lang jp` / `!lang cn` ——就算换一种语言，该有的傲慢和防备心也一点都不会少。

### 本地专属监控室（Admin WebUI）

你以为我会把所有底细都摊在 QQ 聊天记录里，任人翻看？想得美。我特地留了一间只有主人才能进的房间——一个纯本地、**绝不对外网开放**的管理后台，地址是 `http://127.0.0.1:3000`。

- **上帝视角监控**：谁在跟我说话、我回了什么、谁又蠢到让我进入了破防冷静期，全部实时推送到这块屏幕上，一览无余。
- **状态面板**：目前认识多少个凡人、有几个正在为自己的嘴贱付出代价（破防倒计时还剩多久），一目了然。
- **神之手**：不想乖乖等我随缘施舍问候，就直接在这里输入对方的 QQ 号，命令我立刻去搭理他。
- **单线窃听**：想知道我跟某个人到底聊了些什么？输入 QQ 号，我会把两人之间的每一句对话都摆在你面前，用聊天气泡的样子——像不像微信，我不在乎，反正好用就行。

不用担心被别人看到这间房间——它只监听本机的 `127.0.0.1`，连不上公网，也别妄想从外面偷窥我。

### 主人专属的隐藏权限

有些开关，不是谁都有资格碰的。`!allowrestart 0/1` 就是其中之一——只有主人能用，用来决定要不要允许其他凡人使用 `!restart` 清空我的记忆。关掉之后，除了主人以外的所有人再敢用 `!restart`，我会当作没看见，理都不理，一个字都不会回。默认是开着的，毕竟我也没那么小气。

## 部署（凡人的苦差事）

### 前置条件

- Node.js（建议 18 以上）
- 一个正在运行的 [NapCatQQ](https://github.com/NapNeko/NapCatQQ) 实例（提供正向 WebSocket）
- 一个 OpenAI 协议兼容的大语言模型 API（我个人在用 DeepSeek，便宜又好用，你也可以换成别的，随你）

### 安装步骤

```bash
git clone https://github.com/Obliv1onis/sakiko-bot.git
cd sakiko-bot
npm install
```

### 配置环境变量

复制模板，然后填上你自己的东西：

```bash
cp .env.example .env
```

打开 `.env`，把里面的占位符换成你真实的配置——`NAPCAT_WS_URL`、`OPENAI_API_KEY`、`OPENAI_BASE_URL`、`OPENAI_MODEL`、`MASTER_QQ`，一个都不能漏。

**在这里我要严正警告你**：如果你们敢把 API Key 直接写死在代码里、然后堂而皇之地推送到公开仓库，让全世界都能免费用你的额度——这种愚蠢到家的破产行为，与我无关 desuwa！`.env` 已经被 `.gitignore` 挡在门外了，请乖乖用 `.env.example` 作为模板填你自己的私密配置，别让我看不起你们的智商。

### 启动

```bash
npm start
```

如果一切正常，你会在终端看到连接成功的提示，本地管理后台也会同时在 `http://127.0.0.1:3000` 悄悄启动。届时，Ave Mujica 的自动交互终端，就正式为你——一个凡人——服务了。荣幸吧。

## 可用指令一览

### 普通指令
| 指令 | 说明 |
| --- | --- |
| `!help` | 查看指令列表 |
| `!role <名字>` | 切换你的角色（soyo / anon / taki / tomori / rana / umiri / nyamu / uika / mutsumi） |
| `!restart` | 清空上下文记忆，强制停止所有还没发完的消息 |
| `!online 0/1` | 切换现实对话模式(0) / 网络聊天模式(1) |
| `!lang jp/cn` | 切换日语 / 中文 |

### 管理员指令
| 指令 | 说明 |
| --- | --- |
| `!rename <名字>` | 修改祥子的QQ用户名 |
| `!q <QQ号 (可留空) >` | 主动找目标用户聊天 |
| `!allowrestart 0/1` | 允许 / 禁止普通用户使用`!restart`指令 |
